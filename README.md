# Rockcraft Plugin

A repository to learn how to create Minecraft plugins and teach my nephew how to build them as well. 

## Get the Latest Code

To get the latest code run the following command.
`git pull`

## Pushing your code to the server

When you make changes you can push your code to the server with the following commands

1. `git add -A`
2. `git commit -m "Say something about your changs"`
3. `git push`
